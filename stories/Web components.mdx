# Macrostrat web components

Macrostrat's web components are a collection of user interface components for
rendering geologic information. They are heavily used in
[Macrostrat's web application](https://macrostrat.org),
[Rockd](https://rockd.org), and other applications for authoring and exploring
information about the Earth's crust.
